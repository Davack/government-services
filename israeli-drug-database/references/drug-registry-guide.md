# Israeli Drug Registry Guide

## Ministry of Health Drug Registry
- Service page: https://www.gov.il/he/service/israeli-drug-index
- Registry search UI: https://israeldrugs.health.gov.il/
- Public access, searchable
- Covers all approved pharmaceuticals in Israel
- Note: the old `data.health.gov.il` domain is no longer resolvable. Use the gov.il service page as the primary entry point and `israeldrugs.health.gov.il` for the live search UI.

## Key Fields
| Field | Hebrew | Description |
|-------|--------|-------------|
| Trade name | shem miskhari | Brand name |
| Generic name | shem generi | INN name |
| Active ingredient | chomer peeil | Active substance |
| Dosage form | tzurat matan | Tablet, capsule, etc. |
| Manufacturer | yatzran | Marketing authorization holder |
| ATC code | kod ATC | WHO classification |
| Status | matzav rishum | Active/suspended/cancelled |
| Prescription | sug mircham | OTC/Prescription/Restricted |

## Prescription Categories
- OTC (llo mircham): No prescription needed
- Prescription (mircham): Physician required
- Restricted (mircham meyuchad): Specialist only
- Narcotic (sam mefakach): Special form

## Health Basket Tiers
- Full basket, no copay: Essential medications
- Full basket, with copay: ~10-50 NIS per package
- Partial basket: Percentage copay
- Not in basket: Full price or via supplementary insurance

## Generic Drugs
- Typically 40-60% cheaper than brand-name
- Same active ingredient, strength, and form
- Lower copay in health basket
- Switching requires physician approval
